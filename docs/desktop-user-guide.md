# QingTabooFinder 桌面版使用说明

桌面版在本地电脑运行，待查验文档不会上传到网络。下载程序后，按下面的步骤即可开始查验。

## 1. 下载程序

1. 打开项目 GitHub 页面中的 **Releases**。
2. 下载适合自己电脑的压缩包：
   - Windows：`QingTabooFinder-Windows.zip`
   - macOS：`QingTabooFinder-macOS.zip`
3. 解压下载的文件，并保留解压后的完整文件夹。

首次打开时，macOS 或 Windows 可能显示安全提示。确认文件来自本项目的 GitHub Release 后，按下方“常见问题”中的对应步骤打开即可。

## 2. 准备文件

- **待查验文档**：TXT、DOCX 或 EPUB 格式的文本。
- **规则 CSV**：程序自带 `qing_taboo.csv`。一般不需要更换；只有使用自己的规则表时，才通过“选择...”更换。

请只使用自己有权阅读、保存和处理的文本。报告中的命中结果是待复核的线索，不单独构成断代或版本结论。

## 3. 完成一次查验

1. 双击打开 `QingTabooFinder`。
2. 在“待查验文档”一行点击“选择...”，选取需要检查的 TXT、DOCX 或 EPUB 文件。
3. 在“目标皇帝”下拉框中选择研究对象。
4. 勾选需要检查的避讳项目。初次使用建议保留默认的前四项：繁体、简体、偏旁和替代字。
5. 确认“报告输出目录”。默认目录为个人文件夹下的 `QingTabooFinder Reports`。
6. 点击“开始查验”。文档较长时，请等待状态文字显示“查验完成”。
7. 点击“打开输出目录”，查看生成的 CSV 与 Excel 报告。

## 4. 阅读报告

每次查验都会生成同一内容的两份文件：

- `*_避讳查验报告.csv`：适合用 Excel、Numbers 或 WPS 筛选、整理。
- `*_避讳查验报告.xlsx`：适合直接查看。

重点字段包括命中字、命中类别、所属皇帝、全文位置、分段位置和命中上下文。请结合原书影像、刊刻背景、文本性质及历史语境进行复核。

## 5. 常见问题

### macOS 提示无法打开应用

首次打开时，如果 macOS 提示无法验证开发者或阻止打开应用：

1. 先关闭提示窗口。
2. 打开苹果菜单，进入“系统设置”。
3. 选择“隐私与安全性”。
4. 向下滚动到“安全性”，找到关于 QingTabooFinder 被阻止打开的提示。
5. 点击“仍要打开”或“允许打开”，再按提示确认。

在 macOS 26.5.1 等版本中，“仍要打开”通常会在首次尝试打开应用后出现在“隐私与安全性”的“安全性”区域。如果没有看到该选项，先尝试双击打开一次，再回到系统设置查看。也可以在 Finder 中按住 Control 点击 `QingTabooFinder`，选择“打开”，然后在确认窗口中再次选择“打开”。

### Windows 显示 SmartScreen 提示

如果 Windows 显示“Windows 已保护你的电脑”：

1. 确认下载文件来自本项目的 GitHub Release，文件名为 `QingTabooFinder-Windows.zip`。
2. 在提示窗口中点击“更多信息”。
3. 确认发布来源无误后，点击“仍要运行”。

如果电脑由学校、单位或机构管理，且提示由管理员阻止，请联系设备管理员，不要尝试绕过管理策略。

**提示找不到规则文件**：在“规则 CSV”一行点击“选择...”，重新选取 `qing_taboo.csv`。

**没有输出结果**：先确认目标皇帝与所选项目；也可以用项目提供的示例 TXT 验证程序是否正常运行。

**结果过多**：先只勾选基础的繁体、简体、偏旁和替代字，再结合研究问题缩小范围。
